Sponge Privacy Policy

Last updated: March 15, 2026

Overview

Sponge is a lecture recording and AI note-taking app for macOS. We respect your privacy and collect as little data as possible. This policy explains what data is collected and how it is used.

Data That Stays on Your Device

  • Audio recordings — stored locally in the folder you choose. Never uploaded to our servers.
  • Transcripts — generated on-device using Apple's SpeechAnalyzer. Never sent to us.
  • User notes — stored locally via SwiftData. Never sent to us.
  • Gemini API key — stored securely in your Mac's Keychain. Never sent to us.

Data Sent to Third Parties

  • Google Gemini API — When you use AI features (class notes, summaries, recall prompts), your transcript text is sent to Google's Gemini API for processing. This is governed by Google's API Terms of Service. No audio is sent — only the text transcript.
  • TelemetryDeck (anonymous analytics) — We use TelemetryDeck to collect anonymous, privacy-friendly usage statistics. TelemetryDeck does not collect personal data, device identifiers, or IP addresses. It tells us things like how many people use the app and which features are popular. No user content is ever included. See TelemetryDeck's privacy policy.
  • Sparkle (update checks) — The app checks for updates via Sparkle, which fetches an XML file from GitHub Pages. No personal data is sent.

Microphone Access

Sponge requests microphone access to record lectures. Audio is processed and stored entirely on your device. Microphone data is never transmitted to any server.

Data We Do Not Collect

  • We do not collect your name, email, or any personal information.
  • We do not track your location.
  • We do not use advertising trackers or sell data.
  • We do not have user accounts or logins.

Contact

If you have questions about this privacy policy, email sponge@waitworks.com.
