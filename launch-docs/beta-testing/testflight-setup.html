Internal Testing (Immediate Access)

  • Add testers by email/Apple ID
  • No Apple review required
  • Immediate access to builds

External Testing (Requires Review)

For wider testing (requires 24-48hr Apple review)

  • Create testing groups
  • Add testers via email or public link
  • First build requires Apple review
  • Can support up to 10,000 testers
